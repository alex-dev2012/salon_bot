# Документация проекта "Чат-бот для салонов красоты"

### Чат-бот для салонов красоты (далее - Бот) представляет собой небольшой телеграмм-бот, основными задачами которого является:

Со стороны клиента:

1. *Запись клиентов на доступные слоты (оставление контактных данных)* ///>
2. *Отмена записи*
3. *Возможность просмотра работ мастера, краткой информации о мастере*
4. *Просмотр прайс-листа и краткой информации о салоне красоты* ///>

Со стороны салона красоты (главного модератора):

1. *Выдача доступа к админ-панеле мастерам*
2. *Все опции, доступные мастеру (см. далее)*

Со стороны мастера:

1. *Доступ к админ-панеле*
2. *Возможность создания лота для записи* ///>
3. *Возможность заполнения портфолио, работ и прайса мастера*

За дополнительную плату:
4. *Возможность заполнения профиля (с фотографией и краткой информацией о мастере)*

Дополнительные системы (за дополнительную плату):
1. *Проверка на подписку на ТГК салона красоты*
2. *Реферальная программа*
3. *Система промокодов*
4. *Система оплаты и предоплаты*


Стоимость данного бота

Без дополнительных систем - 2.000 рублей (без деплоя).
Без дополнительных систем - 2.100 рублей (с деплоем).

Стоимость дополнительных систем

1. Возможность заполнения профиля мастером - 200 рублей
2. Проверка на подписку - 100 рублей
3. Реферальная программа + Система промокодов - 100 рублей
4. Система оплаты и предоплаты (с интеграцией платежных сервисов) - 500 рублей

Стоимость бота с полным функционалом - 3.000 рублей
